# arduino_testing

This is just a code for Arduino UNO Showing Display to OLED Display 0.96inch

# Requirement
- 0.96inch OLED Display
- 4 pin button
- LED
- 330ohm Resitor
- Arduino UNO

# Connect Pin

| PIN        | OLED           |
| ------------- |:-------------:|
| 5V      | VDD |
| GND      | GND      |
| A5 | SCK      |
| A4 | SDA      |


| PIN        | Component           |
| ------------- |:-------------:|
| D2      | Resitor >> +Anode LED |
| GND     | -Cathode LED |
| D3      | 1b Button |
| GND     | 2a Button |
